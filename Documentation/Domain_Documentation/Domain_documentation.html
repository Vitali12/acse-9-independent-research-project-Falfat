ConvertPolysurfaceToSurface(polysurface)¶

converts polysurfaces to surface and returns False and surface if the argument is a polysurface. Returns True and the argument type if it is not a polysurface. Polysurfaces are made by SplitBrep We can’t use normal functions on them. So, we have to turn them into seperate surfaces with ExplodePolysurfaces. SplitBrep function is used in RemoveSurfacesOutsideOfBox(). An auxilliary method to trim extruding fractures.

ploysurface: GUID
guid of a Rhino surface
